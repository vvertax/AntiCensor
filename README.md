# AntiCensor · [English](README-Eng.md)

Расширение для [Spicetify](https://spicetify.app), которое автоматически заменяет цензурированные треки на версии без цензуры.

![preview](preview.gif)

## Как это работает

При воспроизведении трека расширение проверяет его Spotify ID в удалённой базе данных. Если трек найден — Spotify заглушается, и вместо него воспроизводится чистая версия из базы. Слайдер громкости заменяется на собственный, синхронизированный с воспроизведением.

## Установка

### Через Spicetify Marketplace

Найди **AntiCensor** во вкладке Extensions в Spicetify Marketplace и нажми Install.

### Вручную

```bash
cp AntiCensor.js ~/.config/spicetify/Extensions/
spicetify apply
```

## Использование

- Расширение работает автоматически — ничего настраивать не нужно
- При замене трека появляется уведомление
- Громкость регулируется слайдером, который появляется вместо стандартного во время замены

### Контекстное меню (ПКМ на треке)

| Пункт | Действие |
|---|---|
| Заменить на версию без цензуры | Вручную привязать свой аудиофайл к треку |
| Удалить замену без цензуры | Отключить замену для этого трека |

## База данных

База данных хранится в [`db.json`](db.json) этого репозитория. Аудиофайлы — в папке [`tracks/`](tracks/).

Формат `db.json`:

```json
{
  "tracks": {
    "SPOTIFY_TRACK_ID": "https://raw.githubusercontent.com/vvertax/AntiCensor/main/tracks/FILENAME"
  }
}
```

Spotify Track ID можно найти в ссылке на трек: `open.spotify.com/track/`**`TRACK_ID`**

## Ограничения

- Прогресс-бар и скраббер отображают позицию цензурного трека Spotify
- Если цензурная версия короче оригинала, трек переключится раньше времени
